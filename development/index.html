Development

For this project, I made a reactive interface based on buildings at Brown University. You are able to sort the set of buildings by construction year and area, and are able to filter by use category of building and region of campus. You can keep a list of your favorite buildings.

This was my first project in React, which I enjoyed. I found it rewarding to be able to learn a new way to develop websites.
